# Merch / Drop Sticker Images

| File | Subject | Recommended Size |
|---|---|---|
| `grand-line-sticker.png` | Grand Line Collection sticker pack art | 600 × 600 px (square) |
| `neo-seoul-sticker.png` | Neo-Seoul Collection sticker pack art | 600 × 600 px (square) |

**Tips:**
- Export as PNG with a **transparent background** — the card has a dark green spotlight effect behind it.
- `object-fit: contain` is used so the full sticker shape is visible without cropping.
- A `drop-shadow` filter is applied via CSS, so avoid pre-baking your own shadow into the PNG.
